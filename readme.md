# 貝利瘋狂爆PC金幣MOD
**功能：**  
* **為Degrees-of-Lewdity新增要素：**
  * 貝利租金欠款新增累積制模式，本週沒收到的欠款將累計至下週，逐週累計，直到PC被貝利找到觸發還款事件為止。
